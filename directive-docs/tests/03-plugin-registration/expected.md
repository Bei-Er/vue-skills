# 权限控制(0)

---

## 使用示例

```vue
<template>
  <button v-permission="'admin'">删除</button>
</template>
```

---

## 指令值

| 类型 | 说明 |
|------|------|
| `string` | 权限标识，如 `'admin'`、`'editor'` |

---

## 钩子函数

| 钩子 | 说明 |
|------|------|
| mounted | 根据权限判断是否移除元素 |
| updated | 权限变化时重新判断 |

---

# 防抖

---

## 使用示例

```vue
<template>
  <button v-debounce:500.immediate="handleClick">按钮</button>
</template>
```

---

## 指令值

| 类型 | 说明 |
|------|------|
| `Function` | 防抖后执行的回调函数 |

---

## 指令参数

| 参数 | 说明 |
|------|------|
| `300` | 默认延迟时间（毫秒） |
| `500` | 自定义延迟时间（毫秒） |

---

## 修饰符

| 修饰符 | 说明 |
|--------|------|
| `.immediate` | 立即执行第一次回调 |

---

## 钩子函数

| 钩子 | 说明 |
|------|------|
| mounted | 绑定防抖点击事件 |
| unmounted | 移除事件监听 |
